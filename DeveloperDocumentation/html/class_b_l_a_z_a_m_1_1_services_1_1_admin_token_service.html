Detailed Description

Provides an administrative token. Note that tokens expire after one hour and are static to the server, meaning every user and the server itself gets the same token. This may be considered a security risk, but the lack of any data transfer or server control via the RestAPI mean there is no risk.

If a real RestAPI is implemente this will need to change to jwt tokens.
