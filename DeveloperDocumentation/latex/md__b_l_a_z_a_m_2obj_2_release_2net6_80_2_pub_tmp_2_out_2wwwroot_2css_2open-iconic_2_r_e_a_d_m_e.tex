Open Iconic also comes in a SVG sprite which allows you to display all the icons in the set with a single request. It\textquotesingle{}s like an icon font, without being a hack.

Adding an icon from an SVG sprite is a little different than what you\textquotesingle{}re used to, but it\textquotesingle{}s still a piece of cake. {\itshape Tip\+: To make your icons easily style able, we suggest adding a general class to the} {\ttfamily \texorpdfstring{$<$}{<}svg\texorpdfstring{$>$}{>}} {\itshape tag and a unique class name for each different icon in the} {\ttfamily \texorpdfstring{$<$}{<}use\texorpdfstring{$>$}{>}} {\itshape tag.} ~\newline
