A stateful "{}session"{} store for the application\textquotesingle{}s user session state. This class is a "{}hack"{} for Blazor Server to allow it to behave as if there is a persistent user web session between reloads on the client. Using this class data can be persisted over the course of an entire user\textquotesingle{}s login session. In the context of this app, this class\textquotesingle{}s primary purpose is to cache the user permission for logged in Active\+Directory users. Each logged in user\textquotesingle{}s Claims\+Principal is cached at login, retrieved on reload, and removed either on logout, or after 3x the Timeout set in the Authentication\+Settings in the Database. Of note\+: On webapp restart, if a valid Claims\+Principal still exists in the user\textquotesingle{}s browser cookies for a logged in user when they reload any page the cache is updated with the missing Claims\+Principal.
